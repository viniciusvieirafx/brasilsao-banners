# Brasilsão — Banners

Repositório de imagens para os banners do mundo VRChat **Brasilsão**.

## Como adicionar uma imagem nova

1. Faça upload da imagem neste repositório (JPG ou PNG, 1920×1080)
2. Abra o arquivo `banners.txt`
3. Adicione a URL raw da imagem no final do arquivo:
   ```
   https://raw.githubusercontent.com/viniciusvieirafx/brasilsao-banners/main/nome-da-imagem.jpg
   ```
4. Salve — na próxima vez que o mundo recarregar, a imagem aparecerá.

## Como remover uma imagem

Apague ou comente (com `#`) a linha correspondente no `banners.txt`.

## Dicas

- Resolução ideal: **1920×1080** (16:9)
- Formatos aceitos: JPG, PNG
- Linhas em branco e linhas começando com `#` são ignoradas pelo sistema
